# spacy-python-lib
This repository contains examples demonstrating the usage of the [spaCy](https://spacy.io/) library in Python.

Inside the `lib-course/` directory, you will find several Python scripts organized by chapter, each showcasing different features and capabilities of spaCy, such as:
- Tokenization
- Lexical attributes
- Pipeline and packages
- Part-of-speech tagging
- Named entity recognition

These examples are intended for learning and reference purposes. To get started, make sure to install the required dependencies listed in `requirements.txt`.

## Getting Started

1. Clone this repository.
2. Install dependencies:
	```bash
	pip install -r requirements.txt
	```
3. Explore the scripts in `lib-course/` to see spaCy in action.
